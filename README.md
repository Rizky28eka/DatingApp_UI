# ❤️ DatingApp UI - Flutter Modern Design

A premium, modern, and clean Dating Application UI built with Flutter. This project showcases a high-fidelity user interface design focused on user experience, featuring smooth transitions and a sleek aesthetic.

---

## ✨ Features

- **👤 Interactive Profile Dashboard**: A comprehensive view of user stats, progress, and matches.
- **⭐ Super Likes Me**: Dedicated page for managing premium interests.
- **🔥 Matches Grid**: Dynamic grid view to browse through your connections.
- **💬 Messages & Chat**: Clean and intuitive messaging interface with search functionality.
- **🎨 Premium Style Guide**: Centralized management of colors, typography (Raleway), and common components.

---

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev/)
- **Language**: [Dart](https://dart.dev/)
- **Typography**: [Raleway Font Family](https://fonts.google.com/specimen/Raleway)
- **State Management**: Standard Flutter `StatefulWidget` / `StatelessWidget`

---

## 📂 Project Structure

```bash
lib/
├── common/          # Shared & Reusable UI components
├── pages/           # Main screens (Profile, Matches, Messages, etc.)
├── styleguide/      # Centralized Colors and TextStyles
└── main.dart        # App Entry Point
```

## 🚀 Getting Started

### Prerequisites

- Flutter SDK installed.
- Android Studio / VS Code with Flutter extension.
- An emulator or physical device.

### Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/yourusername/DatingApp_UI.git
   ```

2. **Navigate to project directory**:

   ```bash
   cd DatingApp_UI
   ```

3. **Install dependencies**:

   ```bash
   flutter pub get
   ```

4. **Run the app**:

   ```bash
   flutter run
   ```
