# ❤️ DatingApp UI - Premium Flutter Kit

A high-fidelity, modern, and clean Dating Application UI Kit built with Flutter. Designed for seamless user experience with smooth transitions and a premium aesthetic.

---

## 📖 1. Project Title & Description
**Name**: DatingApp UI  
**Description**: A comprehensive Flutter UI starter kit for dating applications, featuring premium design patterns, smooth animations, and a modular architecture.  
**Goal**: To provide developers with a production-ready UI foundation for social and dating platforms, reducing development time while maintaining high design standards.

---

## 🌐 2. Overview / Background
The modern dating app market requires not just functionality, but an emotional connection through design. This project was created to bridge the gap between complex backend logic and premium user interfaces. It offers a "plug-and-play" UI experience that can be easily integrated with any backend service.

**The Problem**: Many developers struggle to create high-quality, responsive UIs that feel "native" and "premium."  
**The Solution**: A curated collection of common dating app screens (Profile, Matches, Messages) built using Flutter's best practices.

---

## ✨ 3. Features (Fitur Utama)

### 🔐 Authentication (Planned)
- [ ] **Login/Logout**: UI placeholders for secure user access.

### 📊 Dashboard & Discovery
- **Profile Dashboard**: Comprehensive view of user stats and personal info.

### 💬 Messaging
- **Real-time UI**: Intuitive chat interface with search and filter capabilities.

---

## 🛠️ 4. Tech Stack
- **Frontend**: [Flutter](https://flutter.dev/) (Framework) & [Dart](https://dart.dev/) (Language)
- **Design System**: Vanilla CSS-like styling using Flutter's `ThemeData` and custom `StyleGuide`.
- **Typography**: [Raleway](https://fonts.google.com/specimen/Raleway) (Geometric Sans-serif).
- **Icons**: Material Icons & Cupertino Icons.

---

## 🏗️ 5. System Architecture
This project follows a **Modular UI Architecture**:
- **Common Layer**: Reusable widgets (Buttons, AppBars, etc.).
- **Pages Layer**: Discrete screens for specific user flows.
- **StyleGuide Layer**: Centralized tokens for colors and typography to ensure consistency.

---

## 🚀 6. Installation Guide

### Requirements
- Flutter SDK `^2.18.6`
- Dart SDK `^2.18.0`
- Android Studio / VS Code

### Step-by-step
1. **Clone Repo**:
   ```bash
   git clone https://github.com/rizky28eka/DatingApp_UI.git
   ```
2. **Install Dependencies**:
   ```bash
   flutter pub get
   ```
3. **Run Project**:
   ```bash
   flutter run
   ```

---

## 📱 7. Usage
1. **Launch App**: The app starts at the **Profile Page**.
2. **Browse Matches**: Navigate to the Matches tab to see potential connections.
3. **Check Messages**: Access the Messages screen to view active conversations.
